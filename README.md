# 自由回忆系列位置效应实验

## 项目简介

本项目旨在复现 Murdock (1962) 关于自由回忆中系列位置效应的经典实验，验证两种记忆说对首因效应、近因效应及水平渐近线的理论预测。

## 实验设计

### 实验材料
- 汉语双字词列表，均选自日常高频词汇，排除同音词、专有名词及语义关联词
- 六组不同长度和呈现速率的单词列表

### 实验条件
采用 3（总时间类别）× 2（列表长度-呈现速率组合）设计：

| 总时间类别 | 组别 | 列表长度 | 呈现速率 | 总时间(秒) |
| ---------- | ---- | -------- | -------- | ---------- |
| 第一类     | 10-2 | 10       | 2 秒/词  | 20         |
| 第一类     | 20-1 | 20       | 1 秒/词  | 20         |
| 第二类     | 15-2 | 15       | 2 秒/词  | 30         |
| 第二类     | 30-1 | 30       | 1 秒/词  | 30         |
| 第三类     | 20-2 | 20       | 2 秒/词  | 40         |
| 第三类     | 40-1 | 40       | 1 秒/词  | 40         |

### 实验程序
1. 指导语阶段：呈现标准化指导语
2. 列表呈现阶段：按设定速率依次呈现单词
3. 回忆阶段：被试在 1.5 分钟内进行自由回忆
4. 循环与休息：每完成 5 个列表强制休息 30 秒

## 实验结果

### 平均回忆单词数
| 总时间类别 | 组别 | 平均回忆数 M | SD | 组内比较 (p) |
| ---------- | ---- | ---------------- | ------ | ---------------- |
| 第一类     | 10-2 | 6.42             | 0.82   | t(28)=0.32, p=0.75 |
| 第一类     | 20-1 | 6.35             | 0.91   |                  |
| 第二类     | 15-2 | 8.11             | 1.05   | t(28)=0.37, p=0.71 |
| 第二类     | 30-1 | 7.98             | 1.18   |                  |
| 第三类     | 20-2 | 8.74             | 1.21   | t(28)=0.41, p=0.68 |
| 第三类     | 40-1 | 8.58             | 1.35   |                  |

### 系列位置效应
- **首因效应**：前 3-4 个位置回忆概率陡峭下降
- **近因效应**：最后约 8 个位置回忆概率 S 型上升
- **中间渐近线**：位置 5 至倒数第 9 区间回忆概率平稳

### 近因效应的 Gompertz 拟合
所有组的决定系数 r^2 均大于 0.96，拐点 x_i 集中在 1.5-1.6 词之间。

## 结论

1. 自由回忆的系列位置曲线由**首因效应**、**近因效应**和**水平渐近线**三部分构成
2. 总呈现时间相同的列表产生等量的回忆单词数，与列表长度或呈现速率无关
3. 近因效应可用 Gompertz 双指数函数精确描述，拐点位于倒数第 2-3 词之间
